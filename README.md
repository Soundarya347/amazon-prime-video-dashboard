# Amazon Prime Video Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard that analyzes Amazon Prime Video data.

## Key Insights
- Content distribution by genre
- Ratings and popularity trends
- Year-wise content growth
- Country-wise availability

## Tools Used
- Power BI
- Data Cleaning & Transformation

## File
- Amazon Prime Video Dashboard.pbix

## How to Use
1. Download the .pbix file
2. Open in Power BI Desktop
